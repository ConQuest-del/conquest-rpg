Telegram 2FA Authentication

ConQuest uses a single official bot for all players. No BotFather setup required ‚Äî just open Telegram and send one command.

1
GET CODE
Click CONNECT. A unique verification code is generated for your session (e.g. A3F2-C9D1).
2
OPEN BOT
Tap OPEN @ConQuestBot to deep-link directly to the bot in Telegram.
3
SEND CODE
Send /verif A3F2-C9D1 to the bot. The bot acknowledges receipt.
4
ENTER OTP
Bot replies with a 6-digit OTP. Enter it in the game to complete verification.
5
PLAY!
Verified! The bot sends live notifications: level ups, epic drops, legendary boss alerts.
‚úÖ Bot Notifications: Once verified, @ConQuestBot sends you real-time alerts ‚Äî level up, legendary item drops, boss spawns, and periodic status reports ‚Äî directly in Telegram.
